# Features

Create one folder per feature:

```txt
spec/features/001-feature-name/
  spec.md
  plan.md
  tasks.md
  verify.md
```

Feature workflow:

- `spec.md`: what will be built and acceptance criteria.
- `plan.md`: technical approach and files involved.
- `tasks.md`: small ordered implementation tasks.
- `verify.md`: commands run and acceptance criteria validated.

Read only the active feature and the minimal constitution files needed.
